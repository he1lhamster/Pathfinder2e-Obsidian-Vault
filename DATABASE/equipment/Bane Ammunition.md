**Ammunition** any; **Bulk** —
**Activate** <span class="action-icon">1</span> Interact

---
Monster hunters favor bane ammunition that contains a capsule of reagents tailored to a particular type of creature— aberration, animal, beast, dragon, fey, giant, ooze, or both fungus and plant. Each type requires a different formula. When activated bane ammunition hits a target that has a trait matching the selected type, it takes [[Persistent Damage]] in addition to the damage the attack normally deals.
